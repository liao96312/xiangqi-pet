# 视觉规范

本项目是中国象棋桌面陪练小窗，视觉气质偏向：古典、冷峻、竞技、克制、高级。

## 方向关键词

中国风 · 竞技感 · 霸气 · 克制 · 高级 · 压迫感 · 金属感 · 书法刀锋感 · 战报感

## 棋子质感

- 半写实轻拟物，但不做与棋盘割裂的强 3D 效果
- 两方统一使用暖木棋坯，径向渐变营造克制体积感
- 边缘高光 + 内圈刻线 + 轻落地阴影
- 红黑两方通过朱砂字与墨字区分，材质保持一致
- 选中/高亮用光晕和描边，避免重闪烁

## 动画原则

### 时长

| 场景 | 时长 | 曲线 |
|------|------|------|
| 走子移动 | `0.22s` | `cubic-bezier(0.16, 0.88, 0.34, 1)`（带末端减速） |
| 落子回弹 | `0.28s` | `cubic-bezier(0.15, 0.9, 0.25, 1.1)`（轻微过冲） |
| 选中脉冲 | `0.85s` | ease-in-out，无限循环 |
| 吃子火花 | `0.42s` | ease-out，单次 |
| 将军弹字 | `0.82s` | `cubic-bezier(0.18, 0.9, 0.22, 1)` |
| 绝杀弹字 | `1.02s` | 同上，更慢更重 |
| 绝杀横幅 | `1.45s` | ease-out |

### 总则

- 短而明确，不拖沓
- 走子有起、行、落节奏
- 吃子、将军、绝杀加强反馈但保持高级
- 默认启用，支持 `prefers-reduced-motion`

## 信息层级与密度

### 按钮层级

所有按钮共用同一套 transition（`transform 0.12s ease, background 0.12s ease`）、同一套 hover 反馈（`translateY(-1px)` + `--color-bg-card-hover`）、同一套 active 按下反馈（`translateY(1px) scale(0.96)`）。

| 层级 | 类名 | 尺寸 | 用途 |
|------|------|------|------|
| 品牌 | `.brand-chip` | 36×36 方印 | 标题栏品牌入口，最大最重 |
| 窗口图标 | `.icon-button`（标题栏） | 32×32 圆角方形 | 置顶/最小化/隐藏/退出 |
| 主操作 | `.action-button` | 2×3 网格，最小高 46 | 提示/AI应招/悔棋/复盘/翻转/新局 |
| 设置行 | `.setting-row` | 32 高 | 难度/执子/自动陪练，低于主操作 |
| 详情入口 | `.details-toggle` | 34 高 | 展开底部局面分析抽屉 |
| 结果按钮 | `.result-banner button` | 50×34 药丸 | "再来"重开 |

### 卡片层级

所有卡片共用 `--radius-md` 圆角。是否带边框由内容重要性决定。

| 类名 | 内边距 | 边框 | 用途 |
|------|------|------|------|
| `.turn-card` | `10px` | 有 | 当前行棋方、状态与提示，右侧对局台最高层 |
| `.player-strip` | `5px 8px` | 有 | 双方身份、行棋状态、已吃棋子 |
| `.analysis-panel` | `6px 9px` | 有 | 引擎分析，仅在底部详情抽屉展示 |
| `.review-panel` | `5px 8px` | 无（靠背景色区分） | 复盘结果，临时反馈 |
| `.eval-panel` | 无 | 无 | 评分条，紧凑 |
| `.mate-banner` | `5px 9px` | 有 + 渐变背景 | 绝杀提示，强反馈 |
| `.result-banner` | `8px 10px` | 有 | 胜负结果，最强反馈 |

### 提示语层级

所有次要说明文字统一用 `--font-xs` + `--color-text-secondary`，强调用 `--color-text-strong`。

### 主界面（默认 560×620）

- 第一眼：暖木棋盘；上下棋手条紧贴棋盘，显示身份、轮次和已吃棋子
- 第二眼：右侧对局台；当前状态在上，六个高频操作居中，设置项降到底部
- 第三眼：评分条、分析、复盘、走法列表；默认收起为底部详情抽屉
- 深墨外壳只负责压住画面和突出棋盘，不放大面积装饰图

### 桌宠小窗（compact，220×58）

- 只保留：品牌 chip + 单行状态 + 退出按钮
- 不显示：棋盘、工具栏、详情面板、评分条
- 状态条只展示一行：胜负结果或当前提示，超长省略

### 响应式断点

| 条件 | 行为 |
|------|------|
| `max-height: 680px` 且详情已展开 | 棋盘缩到 `284px`，保证分析抽屉可见 |
| `681px ≤ height ≤ 780px` | 棋盘上限 `332px` |
| `max-width: 500px` | 主界面改成单列；操作区变为 3×2；设置项并排；容器可纵向滚动 |
| `max-width: 500px` 且 `max-height: 680px` | 棋盘 `270px`，隐藏棋手条中的弱信息 |

## 不允许的视觉

- 幼态卡通字体 / Q 版图标
- 大红大黑俗气配色（必须用朱砂红 + 墨黑 + 金属灰）
- 重闪烁、廉价爆炸特效
- 棋盘格线虚、飘、太细
- 不同页面看起来像不同产品
